secondsLeftInInterval :: POSIXTime -> Interval POSIXTime -> Maybe Integer #

Given a POSIXTime, and an Interval this function computes the truncated difference in seconds to the UpperBound of the passed Interval. If the Interval does not have a finite UpperBound, or if the given time is past the finite UpperBound the function will return Nothing. Note that this function is not and should not be used on-chain
